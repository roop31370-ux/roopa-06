# Authentication System Styling - Assignment 2

## Project Overview
This project is an enhancement of a 5-page Authentication System. [cite_start]The goal was to transform plain HTML pages into a professional, responsive, and visually appealing application using **Bootstrap 5** and **Custom CSS**[cite: 7]. [cite_start]This project was completed as part of the Fullstack Java Development course at CampusPe[cite: 2, 3].

## Features Integrated
* [cite_start]**Bootstrap 5 Framework:** Implemented for responsive layouts, cards, and form components[cite: 9, 13].
* [cite_start]**Custom Styling:** A unique `styles.css` file was created to add a professional color palette, Google Fonts (Poppins), and smooth hover transitions [cite: 39, 41-48].
* [cite_start]**Responsiveness:** The application is fully tested and functional across Desktop, Laptop, Tablet, and Mobile screens [cite: 49-54].
* [cite_start]**User Interface:** * Centered login cards for better focus[cite: 20].
    * [cite_start]Professional navigation bar on the Dashboard[cite: 35].
    * [cite_start]Interactive buttons with hover effects[cite: 44].
